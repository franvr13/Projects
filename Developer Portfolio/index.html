My Education

Master Java Web/App Developing

The Java Master offers the knowledge and skills necessary to train as a developer of desktop and web applications using Java, in a context of digital transformation. The syllabus covers from fundamentals to advanced Java, including database connection and web development with technologies such as Servlets, JSP, HTML, CSS, JavaScript (including HTML5 and CSS3). The course prepares students to obtain certifications in Java and apply what they have learned in real projects.

Interior Design

Interior Design training provides the skills necessary to create functional and aesthetically appealing spaces. It covers key areas such as the use of color, lighting, furnishings, space layout and material selection, all with a focus on client needs and sustainability. Students develop proficiency in residential, commercial and public interior planning, learning to combine creativity with technical principles to design environments that enhance the quality of life.
